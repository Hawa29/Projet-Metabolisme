# Projet-Metabolisme

## Analyse des flux métaboliques sur un modèle genome-scale avec MetExplore

Ce projet vise à analyser le métabolisme de Ralstonia solanacearum à l’aide d’un modèle métabolique à l’échelle du génome
Les analyses ont été réalisées avec la librairie CobraPy pour les simulations de flux (FBA) et MetExplore 3 pour la visualisation et le mapping des flux et des gènes.

### Contenu du dépôt

#### Notebook/ : notebooks Jupyter contenant l’ensemble des analyses CobraPy

#### Results/ : fichiers de sortie (flux, gènes essentiels, images MetExplore)
