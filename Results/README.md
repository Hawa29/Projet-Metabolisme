Ce dossier contient l’ensemble des fichiers de sortie générés lors de l’analyse du modèle métabolique de Ralstonia solanacearum.  
Il inclut notamment :  

- les fichiers de flux calculés à partir des simulations FBA,  

- les listes de gènes essentiels pour la croissance optimale,  

- les images issues de MetExplore (mapping des flux et des gènes).
